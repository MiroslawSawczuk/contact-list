# Contact List

Contact list app is a recriument task for displaying fetched data from api in list.

## Installation

1. Clone the project.

```bash
git clone https://github.com/MiroslawSawczuk/contact-list.git
```

2. Open cloned project in root directory.

3. Run the scripts.

```bash
npm install
```

```bash
npm start
```
